Adding and editing places

When creating, you may specify the following data:

  • Name
  • Description
  • Coordinates
  • Type

The name should be short and should match the name of the real-life location make it easy for the clients to decide if the arrangement suits their needs.

The description is longer and gives more context to the client as to what this place entails.

You can enter the coordinates manually through their latitude and longitude, or by double-cliking the map. Be as accurate as possible, but worry too much if you miss by a couple of degrees.

Finally, the type determines which type of activity this place represents. YouTravel does not care about these types - they are mainly there to aid in searching and giving the users a visual aid.

When you're done, click Save and your changes will automatically be updated.


Editing an existing place follows the exact same procedure, and you're free to edit a place at any time.
