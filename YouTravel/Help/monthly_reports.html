Monthly reports

The monthly reports page contains a paginated table with relevant information for the selected month's reservations.

The table contains the following columns:

  • Id: The identification number of the reserved arrangement
  • Reserved by: The username of the client that made a reservation for the arrangement
  • Name: Name of the reserved arrangement
  • Reserved on: Date on which the reservation was made
  • People: Number of people on the reservation
  • Paid?: Payment status of the reservation

Note the icons in the Name column - clicking on those leads us to the details page of that row's arrangement.
