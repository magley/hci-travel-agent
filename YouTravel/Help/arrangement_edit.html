Editing an Arrangement

You can edit all data of an existing arrangement as long until the first reservation for that arrangement is made. Each tab holds a different group of data.

  1. Basic Information

    You can edit the name, description and price of the arrangement on this tab.

  2. Duration

    Set the duration of the arrangement here.

  3. Image

    This is where you can change the image.

  4. Places

    Change the activities (places) here. Double click on the map to add a new place.

After clicking on the submit changes button, the arrangement will be automatically updated.
