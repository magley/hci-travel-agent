Creating an Arrangement

To create a new arrangement, just follow the steps laid out by the YouTravel wizard.

  1. Basic Information

    The name is the first thing our clients will read when glancing over this arrangement. Make it short and sweet! The description can be seen after clicking on an arrangement. You can set the price (in Euros) on here, as well.

  2. Duration

    Set the start and end date for this arrangement using the calendar. Note that unlike other data, you have to set the duration before moving on.

  3. Image

    The image is probably the first thing our clients will notice in general. Clicking on the button on the right opens a file picker where you can select any image (PNG, JPG, JPEG).

  4. Places

    Choose any place from the database as an activity of this arrangement. You can drag-and-drop list items from the top list to the bottom list, or manually move them using the two buttons.
    If you wish to add a new place into the database, double-click on the map, it will compute the coordinate based on your mouse cursor.

  5. Confirm

    Confirm that everything looks good, and click Finish to save the changes.

And that's it! You've made a new arrangement. You'll be navigated to the dashboard where you can see your new arrangement.
