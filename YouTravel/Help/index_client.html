File Menu

Most pertinent to your usage of YouTravel as a guest, the file menu contains the login and register items

Toolbar

The toolbar contains all primary actions you can perform. These include:

  • Navigation
  • Viewing upcoming arrangements
  • Viewing your travel history (only for logged in users)
