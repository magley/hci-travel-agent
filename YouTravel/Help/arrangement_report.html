Arrangement report

On this page you can see a table displaying relevant information for all reservations made for the selected arrangement

Above the table is the name of the arrangement we're looking at. The table contains the following columns:

  • Id: The identification number of the reserved arrangement
  • Reserved by: The username of the client that made a reservation for the arrangement
  • Reserved on: Date on which the reservation was made
  • Paid?: Payment status of the reservation
