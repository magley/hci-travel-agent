Arrangement

An arrangement is an offering for a trip.

When our clients browse the YouTravel catalogue, what they will see are all the active travel arrangements. The client can book any arrangement, making a reservation for traveling.
Think of an arrangement as a class and reservations as instances.

An arrangement has general information - like the name, description and price - but also a collection of places. Each place in an arrangement describes a single activity available to all clients booking said arrangement.

Image 1 shows the main screen for arrangements. This is where you can get an overview of all arrangements, with support for searching and filtering.
You can open this screen at any time by clicking on the View Arrangements toolbar button, opening View > Arrangements, or hitting the CTRL + A hotkey.

When you hover over any arrangement in the list, buttons for removing, viewing statistics and editing the arrangement will become visible.

You can add a new arrangement by clicking on the New button.
